How to Prepare Your Images

1. Create a main folder on your computer (e.g., "My Product Photos")
2. Inside it, create subfolders — name each subfolder as your Product ID
3. You can use any ID format: 100001, BRK-2024-005, TOYOTA-PADS-01, etc.
4. Put your product photos in each subfolder (maximum 5 per product)
5. Select the main folder here and compress
6. Upload the downloaded ZIP to kuruma.lk
